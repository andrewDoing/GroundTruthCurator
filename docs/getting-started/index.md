# Getting Started

Welcome to Ground Truth Curator! This guide will help you set up and start using the platform.

## Overview

Ground Truth Curator consists of two main components:

- **Backend**: FastAPI-based REST API with Python
- **Frontend**: React application built with Vite and TypeScript

## Prerequisites

Before you begin, ensure you have:

- **Python 3.11+** with [uv](https://docs.astral.sh/uv/) package manager
- **Node.js 18+** with npm
- **Jujutsu (jj)** for version control (optional, but recommended for development)
